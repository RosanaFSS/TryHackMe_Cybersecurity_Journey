<h2>Conclusion</h2>
<p>In this room, we learned how computers store information in databases and how SQL is used to ask clear questions about that information. Using a café example, we saw how tables organise orders into rows and columns, and how simple SQL commands can show, filter, sort, and add data.</p>

<h3>SQL Queries Learned in This Room</h3>
<p>
  
- SELECT – choose what data to display<br>
- FROM – choose where the data comes from<br>
- WHERE – filter records based on a condition<br>
- ORDER BY – sort results</p>
