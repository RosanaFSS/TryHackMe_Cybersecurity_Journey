<p>What does a port represent in networking?<br>  
⬛ &nbsp;&nbsp; A method of data encryption.<br>
✅ &nbsp;&nbsp; A specific service or application on a host.<br>
⬛ &nbsp;&nbsp; A network security device.<br>
⬛ &nbsp;&nbsp; A physical network cable connection.<br>
⬛ &nbsp;&nbsp; A type of network protocol.<br>
❯❯❯❯ &nbsp;&nbsp; Ports are used to identity specific services or applications running <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; on a host, allowing multiple services to run on the same IP address.</p>
